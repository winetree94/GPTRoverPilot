Activate Environment

플랫폼

셸

가상 환경을 활성화하는 명령

POSIX - bash/zsh
```
$ source <venv>/bin/activate
```

fish
```
$ source <venv>/bin/activate.fish
```

csh/tcsh
```
$ source <venv>/bin/activate.csh
```

PowerShell
```
$ <venv>/bin/Activate.ps1
```

Install Packages
```
pip install -r ./requirements.txt
```
